Sangpt – AI Landing Page with Liquid Glass UI (Standalone App Plan)

Overview

Create a premium, pixel-perfect landing page for Sangpt, Dev-Link’s AI application.

Focus: liquid glass (glassmorphism) UI throughout

Features: semi-transparent surfaces, backdrop blur, soft gradients, smooth motion, modular sections

Goal: convey a professional, modern, and high-tech feel for the AI application

Fully responsive and scalable to multi-page site
